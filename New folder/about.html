Git Page

Github is a provider of Internet hosting for software development and version control using Git. It allows you to collaborate with others to work on programming projects from anywhere. The main features are: Creating and using repositories Starting and making new branches Make changes and push them as commits Implement and merge changes using pull requests

Repositories:

A repository is used to organize a project. It can contain folders, files, images, spreadsheets or anything related to the project. It is recommended to add a readme file that includes some information about the project.

Branches:

Branches are a way to work on new features for a project without disturbing the main repository.

Edit Files:

You can directly edit files on Github and either commit directly to the main repository or create a branch and open a pull request.
