Azure Page

Microsoft Azure is a cloud computing platform for building, testing, deploying, and managing applications and services through Microsoft-managed data centers. Azure allows for clients to develop and deploy software without having to worry about the hardware and infrastructure. Any issues with the operating systems, servers and networks are also taken care of.

What can you do:

With Azure you can: Create a Virtual Machine, Build a Web Application, Use fully managed SQL Databases, And more, For this tutorial you just need to make an account with Azure and have it ready to deploy images.
