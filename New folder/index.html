Azure Cloud Website Tutorial

In this tutorial, you will get a basic understanding of Github, VI(visual editor), Docker, Azure, and basic terminal Linux commands to create a web page and host it on the Azure cloud service so it can be accessed by anyone anywhere. To begin, it is important to have an understanding of GitHub because it is the main place where you can create and edit projects when working with other people. It is not required in this tutorial so click on the GitHub tab to learn more.

Tutorial

Part 1:

To begin, you need to create a github repository for your project and collaborate with your partners to make your website.

Step 2:

The next step is to download the repository so you have it locally on your machine.

Step 3:

After downloading the project repository, you need to create docker hub repository and merge your project files.

Step 4:

Now that we have our project in our docker hub repository, we need to setup Azure. First open a Powershell terminal and run the following commands:

docker login azure (this will open your browser and allow you to login to Azure)

docker context create aci myaci (it will then ask which subscription you would like to use if you have more than 1. Then you can choose which resource group to use or create a new one.)

docker context list (this will list your aci contexts to verify that yours is up and running.)
