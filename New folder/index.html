Git Page

In this tutorial, you will get a basic understanding of Github, VI(visual editor), Docker, Azure, and basic terminal Linux commands to create a web page and host it on the Azure cloud service so it can be accessed by anyone anywhere. To begin, it is important to have an understanding of GitHub because it is the main place where you can create and edit projects when working with other people. It is not required in this tutorial so click on the GitHub tab to learn more.
