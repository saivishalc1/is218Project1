Linux Commands

Linux is an operating system. Linux is an entire family of operating systems that are based on the Linux Kernel. Linux was released in 1991, and it has begun to grow in popularity because of its ability to be open source. So when using Linux you are going to need a shell, which gives you access to operating system services.

Commands

Some of the most important Linux commands are listed here below.

List of all the commands:

Pwd - It stands for "Print Working Directory", and it will tell you which file you are on.

ls - This lists out all the files that are you in your directory

cd - This allows you to move to a different directory

mkdir - Creates a directory

mv - Lets you move files to another folder

touch - Creates new files

vi - open an existing file

git init - creates a new repo

git init - creates a new repo

git commit -m "Message" - Allows you to put down a message when initializing a repo

git checkout - Let's navigate between the branches

docker compose up - starts docker and keeps it running

docker compose down - stops the docker from running and kills it
