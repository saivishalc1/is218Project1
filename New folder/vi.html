VI (Visual Editor)

VI stands for “visual editor” and is the default text editor that comes with the UNIX operating system. It is now pretty outdated by today’s standards but it still remains an essential tool for command line file creation and editing.

Getting Started

To begin using VI, run the following command: $ vi filename

Getting Started: Part 2

The two modes of operation within Vi are: Command mode - commands which cause action to be taken on the file, and Insert mode - in which entered text is inserted into the file.
